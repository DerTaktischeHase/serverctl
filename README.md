# serverctl

`serverctl` ist ein SSH-freundliches Wartungs- und Status-Dashboard für Linux-Server.
Es bündelt wichtige Informationen wie Systemstatus, Temperatur, RAM, Disk, Netzwerk, offene Ports und Service-Status.

## Installation (empfohlen – transparent & sicher)

```sh
curl -fsSL https://raw.githubusercontent.com/DerTaktischeHase/serverctl/main/install.sh | bash
```

## Voraussetzungen

- Linux (Debian/Ubuntu empfohlen, funktioniert "best effort" auch auf anderen Distros)
- SSH-Zugang (optional, aber typischer Use-Case)
- `bash`
